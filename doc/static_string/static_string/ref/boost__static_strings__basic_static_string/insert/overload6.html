basic_static_string::insert (6 of 15 overloads)

Insert a string.

Synopsis 
template<
    std::size_t M>
BOOST_STATIC_STRING_CPP14_CONSTEXPR basic_static_string&
insert(
    size_type index,
    const basic_static_string< M, CharT, Traits >& str,
    size_type index_str,
    size_type count = npos);
Description

Inserts a string, obtained by str.substr(index_str, count) at the position index.

Exception Safety

Strong guarantee.

Remarks

The insertion is done unchecked when the capacity of str differs from that of the string the function is called on.

Remarks

All references, pointers, or iterators referring to contained elements are invalidated. Any past-the-end iterators are also invalidated.

Type

Description

Description

The index in str to start inserting from.

The number of characters to insert. The default argument for this parameter is basic_static_string::npos.
