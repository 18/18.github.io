basic_static_string::reserve

Reserve space for n characters, excluding the null terminator.

Synopsis 
BOOST_STATIC_STRING_CPP14_CONSTEXPR void
reserve(
    size_type n);
Description

This function has no effect when n <= max_size().

Exceptions

Type

Thrown On

std::length_error

if n > max_size()
